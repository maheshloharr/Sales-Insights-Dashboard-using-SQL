# 📊 Sales Insight Dashboard

An interactive **Power BI dashboard** designed to analyze sales performance, customer behavior, and order insights using SQL and data visualization techniques.

---

## 🚀 Project Overview

This project provides a comprehensive view of business performance by transforming raw sales data into actionable insights.

It helps answer:

- 💰 How much revenue is generated over time?  
- 👥 How many customers and orders are there?  
- 🛍️ Which products perform best?  
- 🚚 What is the order delivery status?  
- 🌍 How are sales distributed geographically?  

---

## 🧩 Features

- 🎯 Interactive Filters:
  - Gender  
  - State  
  - Category  

- 📊 Key Metrics:
  - Total Revenue 💰  
  - Total Customers 👥  
  - Total Orders 📦  
  - Average Order Value 📈  

- 📈 Visual Insights:
  - Revenue by Year  
  - Revenue by Product  
  - Sales by State (Map Visualization)  
  - Revenue by Gender  

- 🚚 Order Analysis:
  - Delivered Orders  
  - Returned Orders  
  - Delayed Orders  
  - In-Transit Orders
